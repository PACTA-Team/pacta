## SQL Injection Test

**Test:** `POST /api/auth/login` with SQL injection payload
```json
{"email": "' OR 1=1 --", "password": "anything"}
```
**Result:** 401 `{"error":"user not found"}`
**Status:** ✅ PASS — Parameterized queries prevent SQL injection
