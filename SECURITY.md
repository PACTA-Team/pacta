# Security Policy

## Reporting a Vulnerability

We take security seriously. Please report vulnerabilities privately to:

- Email: security@pacta.local
- PGP Key: Available upon request

We aim to respond within 72 hours and fix within 90 days.

## Supported Versions

Only the latest minor version receives security updates.

## Security Updates

We regularly publish security updates and advisories via our GitHub Security Advisories page.

## Bug Bounty

We do not currently have a bug bounty program, but we appreciate responsible disclosure.
